##Intall SCCM Client Manully:
	1. Copy Client Folder From SCCM Server.
		\\<FQDN_Site_Server>\SMS_<SiteCode>\Client
	2. Open Client Foder and run cmd as administrator.
	3. Run ccmsetup with smssite code and smsmp
		ccmsetup.exe SMSSITECODE=<Site_Code> SMSMP=<FQDN_Site_Server>
		ccmsetup.exe SMSSITECODE=PRI SMSMP=sccm.contoso.com

or
\\sccm.lab.com\SMS_S01\Client\ccmsetup.exe SMSSITECODE=S01 SMSMP=sccm.lab.com /logon
	#First login as an admin user.

#Insall SCCM Client Manually in a non-domain PC:

		#provide  password for the mentioned [Administrator] admin user

or
\\sccm.lab.com\SMS_S01\Client\ccmsetup.exe SMSSITECODE=S01 SMSMP=sccm.lab.com /logon
	#First login as an admin user. 

	2. Wait some time then check hostname in sccm device. Than aprove that device.
	3. Check BTS, SMS Agent Host, WMI servies in host machine. Restart them in Client machine.
	4. Run Application Deployment Evaluation Cycle, Machine Policy Retrieval & Evaluation Cycle from Configuration Manager Action.
	5. Allow SCCM-related port in the firewall. Run the SccmAddFirewallPort.ps1 script on the client machine.
